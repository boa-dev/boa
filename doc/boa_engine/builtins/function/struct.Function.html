pub struct Function {
    kind: FunctionKind,
    realm: Realm,
}
Expand description

Boa representation of a Function Object.

FunctionBody is specific to this interpreter, it will either be Rust code or JavaScript code (AST Node).

https://tc39.es/ecma262/#sec-ecmascript-function-objects
