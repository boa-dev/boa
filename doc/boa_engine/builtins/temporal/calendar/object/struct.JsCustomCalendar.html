pub(crate) struct JsCustomCalendar {
    calendar: JsObject,
}
Expand description

A user-defined, custom calendar that is only known at runtime and executed at runtime.

A user-defined calendar implements all of the CalendarProtocolMethods and therefore satisfies the requirements to be used as a calendar.
