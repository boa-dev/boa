Abstract operation CanonicalizeLocaleList ( locales )

Converts an array of JsValues containing structurally valid Unicode BCP 47 locale identifiers into their canonical form.

For efficiency, this returns [Locale]s instead of Strings, since Locale allows us to modify individual parts of the locale without scanning the whole string again.
