Expand description

A Well-Known Intrinsic Object.

Well-known intrinsics are built-in objects that are explicitly referenced by the algorithms of the specification and which usually have realm-specific identities.

Required Methods§

source

fn init(realm: &Realm)

Initializes the intrinsic object.

This is where the methods, properties, static methods and the constructor of a built-in must be initialized to be accessible from ECMAScript.

source

fn get(intrinsics: &Intrinsics) -> JsObject

Gets the intrinsic object.
