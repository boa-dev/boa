pub struct Case {
    condition: Expression,
    body: StatementList,
}
Expand description

A case clause inside a Switch statement, as defined by the spec.

Even though every Case body is a StatementList, it doesn’t create a new lexical environment. This means any variable declared in a Case will be considered as part of the lexical environment of the parent Switch block.
