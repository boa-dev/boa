The opcodes of the vm.

Variants

Pop

Pop the top value from the stack.

Operands:

Stack: value =>

PopIfThrown

Pop the top value from the stack if the last try block has thrown a value.

Operands:

Stack: value =>

Push a copy of the top value on the stack.

Operands:

Stack: value => value, value
