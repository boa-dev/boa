Virtual Machine.

Fields§

§frames: Vec<CallFrame>§stack: Vec<JsValue>§return_value: JsValue§pending_exception: Option<JsError>

When an error is thrown, the pending exception is set.

If we throw an empty exception (None), this means that return() was called on a generator, propagating though the exception handlers and executing the finally code (if any).

See ReThrow and ReThrow opcodes.

This is also used to eliminates crate::JsNativeError to opaque conversion if not needed.
