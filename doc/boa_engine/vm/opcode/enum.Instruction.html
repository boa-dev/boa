Expand description

This represents a VM instruction, it contains both opcode and operands.

Variants§

§

Pop

Pop the top value from the stack.

Operands:

Stack: value =>

§

Dup

Push a copy of the top value on the stack.

Operands:

Stack: value => value, value

§

Swap

Swap the top two values on the stack.

Operands:

Stack: second, first => first, second

§

RotateLeft

Fields

§n: u8

Rotates the top n values of the stack to the left by 1.

Equivalent to calling slice::rotate_left with argument 1 on the top n values of the stack.

Operands: n: u8

Stack: v[n], v[n-1], … , v[1], v[0] => v[n-1], … , v[1], v[0], v[n]

§

RotateRight

Fields

§n: u8

Rotates the top n values of the stack to the right by 1.

Equivalent to calling slice::rotate_right with argument 1 on the top n values of the stack.

Operands: n: u8

Stack: v[n], v[n-1], … , v[1], v[0] => v[0], v[n], v[n-1], … , v[1]
