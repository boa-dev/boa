Expand description

The Operation trait implements the execution code along with the identifying Name and Instruction value for an Boa Opcode.

This trait should be implemented for a struct that corresponds with any arm of the OpCode enum.

Execute opcode with VaryingOperandKind::U16 sized VaryingOperands.
