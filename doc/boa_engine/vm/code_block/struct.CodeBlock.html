Expand description

The internal representation of a JavaScript function.

A CodeBlock is generated for each function compiled by the ByteCompiler. It stores the bytecode and the other attributes of the function.

The [[ClassFieldInitializerName]] internal slot.

§function_environment_push_location: u32

Marks the location in the code where the function environment in pushed. This is only relevant for functions with expressions in the parameters. We execute the parameter expressions in the function code and push the function environment afterward. When the execution of the parameter expressions throws an error, we do not need to pop the function environment.

§parameters_env_bindings: Option<usize>

The number of bindings in the parameters environment.

§trace: Cell<bool>

Trace instruction execution to stdout.
