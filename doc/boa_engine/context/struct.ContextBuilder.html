Expand description

Builder for the Context type.

This builder allows custom initialization of the Interner within the context. Additionally, if the intl feature is enabled, ContextBuilder becomes the only way to create a new Context, since now it requires a valid data provider for the Intl functionality. The required data in a valid provider is specified in BoaProvider

pub fn icu_provider(
    self,
    provider: BoaProvider<'_>
) -> Result<ContextBuilder<'_, 'hooks, 'queue>, LocaleTransformError>

Provides an icu data provider to the Context.

This function is only available if the intl feature is enabled.

Errors

This returns Err if the provided provider doesn’t have the required locale information to construct both a LocaleCanonicalizer and a LocaleExpander. Note that this doesn’t mean that the provider will successfully construct all Intl services; that check is made until the creation of an instance of a service.
