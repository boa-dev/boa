Syntax

  • MDN documentation

Type

More information:

  • ECMAScript reference
  • MDN documentation
§

Uri

An error thrown when the encodeURI() and decodeURI() functions receive invalid parameters.

More information:

  • ECMAScript reference
  • MDN documentation
§

RuntimeLimit

Error thrown when a runtime limit is exceeded. It’s not a valid JS error variant.
