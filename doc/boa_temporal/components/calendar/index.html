Module boa_temporal::components::calendar

source ·
Expand description

This module implements the calendar traits and related components.

The goal of the calendar module of boa_temporal is to provide Temporal compatible calendar implementations.

The implementation will only be of calendar’s prexisting calendars. This library does not come with a pre-existing CustomCalendar (i.e., an object that implements the calendar protocol), but it does aim to provide the necessary tools and API for implementing one.
