Struct boa_temporal::components::Duration

source · 
pub struct Duration {
    date: DateDuration,
    time: TimeDuration,
}
Expand description

The native Rust implementation of Temporal.Duration.

Duration is made up of a DateDuration and TimeDuration as primarily defined by Abtract Operation 7.5.1-5.
