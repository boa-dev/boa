pub unsafe fn from_raw(ptr: NonNull<GcBox<T>>) -> Self

Constructs a Gc<T> from a raw pointer.

The raw pointer must have been returned by a previous call to Gc<U>::into_raw where U must have the same size and alignment as T.

Safety

This function is unsafe because improper use may lead to memory corruption, double-free, or misbehaviour of the garbage collector.
