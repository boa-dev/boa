Struct boa_gc::pointers::gc::NonTraceable

source · 
pub(crate) struct NonTraceable(());
Expand description

Zero sized struct that is used to ensure that we do not call trace methods, call its finalization method or drop it.

This can only happen if we are accessing a GcErasedPointer directly which is a bug. Panics if any of it’s methods are called.

Note: Accessing the crate::internals::GcHeader of GcErasedPointer is fine.
