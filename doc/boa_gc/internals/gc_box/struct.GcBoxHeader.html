Expand description

The GcBoxheader contains the GcBox’s current state for the Collector’s Mark/Sweep as well as a pointer to the next node in the heap.

These flags include:

  • Root Count
  • Mark Flag Bit
  • Weak Flag Bit

The next node is set by the Allocator during initialization and by the Collector during the sweep phase.
