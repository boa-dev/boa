Enum boa_interner::JStrRef

source · 
pub enum JStrRef<'a> {
    Utf8(&'a str),
    Utf16(&'a [u16]),
}
Expand description

An enumeration of all slice types Interner can internally store.

This struct allows us to intern either UTF-8 or UTF-16 str references, which are the two encodings Interner can store.
