Binary expression nodes.

A Binary expression comprises any operation between two expressions (excluding assignments), such as:

  • Logic operations (||, &&).
  • Relational math (==, <).
  • Bit manipulation (^, |).
  • Arithmetic (+, %).
  • The comma operator (,)
