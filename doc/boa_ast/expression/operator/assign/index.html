source · [−]
Expand description

Assignment expression nodes, as defined by the spec.

An assignment operator assigns a value to its left operand based on the value of its right operand. Almost any LeftHandSideExpression Parse Node can be the target of a simple assignment expression (=). However, the compound assignment operations such as %= or ??= only allow “simple” left hand side expressions as an assignment target.
