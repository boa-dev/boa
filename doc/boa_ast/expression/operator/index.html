Operator expression nodes.

An operator expression is an expression that takes an operator (such as +, typeof, +=) and one or more expressions and returns an expression as a result. An operator expression can be any of the following:

  • A Unary expression.
  • A Update expression.
  • A Assign expression.
  • A Binary expression.
  • A Conditional expression.
