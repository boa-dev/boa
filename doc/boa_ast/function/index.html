Functions and classes nodes, as defined by the spec.

Functions are mainly subprograms that can be called by external code to execute a sequence of statements (the body of the function). Javascript functions fall in several categories:

  • Functions.
  • ArrowFunctions.
  • AsyncArrowFunctions.
  • Generators.
  • AsyncFunctions.
  • AsyncGenerators.

All of them can be declared in either declaration form or expression form, except from ArrowFunctions and AsyncArrowFunctions, which can only be declared in expression form.

This module also contains Classes, which are templates for creating objects. Classes can also be declared in either declaration or expression form.
