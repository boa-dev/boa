source

pub fn is_labelled_function(&self) -> bool

Abstract operation IsLabelledFunction.

This recursively checks if this Statement is a labelled function, since adding several labels in a function should not change the return value of the abstract operation:

l1: l2: l3: l4: function f(){ }

This should return true for that snippet.
