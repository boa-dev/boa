pub enum IterableLoopInitializer {
    Identifier(Identifier),
    Access(PropertyAccess),
    Var(Variable),
    Let(Binding),
    Const(Binding),
    Pattern(Pattern),
}
Expand description

A for-in, for-of and for-await-of loop initializer.

The spec specifies only single bindings for the listed types of loops, which makes us unable to use plain LexicalDeclarations or VarStatements as initializers, since those can have more than one binding.
