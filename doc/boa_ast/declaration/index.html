The Declaration Parse Node, as defined by the spec.

ECMAScript declarations include:

  • Lexical declarations (let, const).
  • Function declarations (function, async function).
  • Class declarations.

See Difference between statements and declarations for an explanation on why Declarations and Statements are distinct nodes.
