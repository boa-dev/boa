Traverse the entries in a tree and its subtrees in post or pre order. The callback function will be run on each node of the tree thatâ€™s walked. The return code of this function will determine how the walk continues.

libgit requires that the callback be an integer, where 0 indicates a successful visit, 1 skips the node, and -1 aborts the traversal completely. You may opt to use the enum TreeWalkResult instead.
